{# ── Section 3: Join by Code ── #}
qr_code

Join by Code (Alternative Method)

Students can join individually using your access code or QR code — no Excel needed.

key

Access Code

Your school has a unique Access Code visible on your dashboard. Share it with students. They enter it on the SESA homepage under Join with Code and their account is automatically linked to your school.

qr_code_scanner

QR Code

Your dashboard shows a QR Code students can scan with their phone. This takes them directly to the sign-up page pre-linked to your school — no typing required. Print it and display it in your school.

tips_and_updates

When to use each method: Use Excel upload for bulk onboarding at the start of term. Use access code / QR code for new students joining mid-term or for schools that prefer self-registration.
